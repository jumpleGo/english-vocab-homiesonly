## 🎬 Adjectives for Describing Films

| Word | Meaning (RU) |
|---|---|
| overrated | переоценённый |
| hackneyed | банальный, избитый, заезженный |
| disjointed | несвязный, рваный |
| far-fetched | надуманный, притянутый за уши |
| risqué | откровенный, пикантный |
| harrowing | душераздирающий, мучительный |
| understated | сдержанный, тонкий, без пафоса |

**Examples:**
- *The plot was far-fetched and disjointed.*
- *The acting was understated but brilliant.*

## 🧑 Adjectives for Describing Personality

| Word | Meaning (RU) |
|---|---|
| impetuous | импульсивный, действует не думая |
| taciturn | немногословный, молчаливый |
| gullible | доверчивый, легко обманывается |
| obstinate | упрямый, несговорчивый |
| aloof | отстранённый, держится особняком |
| disdainful | высокомерный, презрительный |

**Examples:**
- *He's so impetuous — he never thinks before acting.*
- *She seems aloof, but she's actually very kind.*
- *Don't be so gullible — not everything on the internet is true.*
