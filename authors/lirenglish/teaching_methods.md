# 🎓 Teaching Methods & Exercise Formats — @lirenglish

> Полезно как для учителей, так и для самостоятельного изучения — многие форматы можно адаптировать для self-study

---

## Exercise Formats

| Format | Description |
|---|---|
| **"How well do you know your partner?"** | Парное задание на Past Simple/Present Perfect — задавать вопросы о партнёре |
| **Lexical Bingo** | Самому нарисовать бинго-карточку и стараться использовать новые слова в речи естественно |
| **Jeopardy / "Своя игра"** | Игровой формат повторения лексики |
| **Paraphrasing drill** | Преподаватель упрощает фразу → ученик вспоминает "крутой" оригинал |
| **3-2-1 method** | Метод для развития скорости речи (говоришь на тему 3 мин, потом 2, потом 1) |
| **% accuracy tracking** | Таблица "уже хорошо / можно улучшить" + % грамматически верной речи отмечается зелёным |

## Memory Techniques (4 techniques from the channel)

1. **Noticing** — замечать новые слова/конструкции в контексте, не выписывая всё подряд
2. **Pick 3-5 words** — выбирать всего 3-5 новых слов с урока/текста для активного заучивания (не больше — для реалистичности)
3. **Active-passive sorting** — делить словарь на «активный» (использую) и «пассивный» (узнаю при чтении/слушании)
4. **Revision at the start of lesson** — начинать каждый урок с повторения предыдущей лексики

## Delayed Correction (отложенное исправление)

Авторский метод: не прерывать ученика во время свободной речи для исправления ошибок. Вместо этого:
1. Записывать ошибки незаметно во время разговора
2. После того как ученик закончил высказывание — разобрать ошибки вместе
3. Это сохраняет уверенность ученика и поток речи, при этом ошибки разбираются

**Видео на YouTube:** "Error correction. Как исправлять ошибки учеников" (@lir_in_english)

## Flipped Classroom

Грамматика — самостоятельно дома (видео, учебник), а время урока полностью посвящается speaking practice и применению грамматики в разговоре. Экономит время урока для самого ценного — живой практики.

---

## Building Speaking Confidence — General Philosophy

- Не гнаться за сложными синонимами в ущерб уверенности — простые слова (good, interesting, I think) — это нормально
- Акцент — не недостаток, а часть identity; цель — понятность, а не "нейтивность"
- Развивать длинные предложения постепенно: простые → с союзами → с relative clauses
- Использовать reported speech по-разговорному (he was like / he went) вместо строгого согласования времён в casual речи
