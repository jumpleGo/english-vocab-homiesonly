## 1. Verb + 过 (guo) — Experience Marker [HSK 2]

Маркер пережитого опыта — "делал когда-то в жизни", без привязки к конкретному времени.

```
看过   — смотрел (когда-то)
去过   — был / ездил (когда-то)
吃过   — ел / пробовал (когда-то)
```

**Examples:**
- 我去过上海。 (Wǒ qù guo Shànghǎi.) — Я был в Шанхае.
- 你吃过火锅吗？ (Nǐ chī guo huǒguō ma?) — Ты пробовал хого?

**Key point:** 过 подчёркивает факт наличия опыта, а не конкретное время действия.

## 2. A + 没有 + B + Adjective — Comparison [HSK 2]

Конструкция сравнения: "A не такой [прил.], как B".

```
我没有他高。           (Wǒ méiyǒu tā gāo.) — Я не такой высокий, как он.
这家店没有那家店便宜。  (Zhè jiā diàn méiyǒu nà jiā diàn piányí.) — Этот магазин не такой дешёвый, как тот.
```

---

## 3. 都 (dōu) — Universal Quantifier / Emphasis

"Все, всё; оба" — усиливает полноту охвата.

```
我们都很忙。     (Wǒmen dōu hěn máng.) — Мы все очень заняты.
她什么都懂。     (Tā shénme dōu dǒng.) — Она понимает всё (без исключения).
希望人人都有。   (Xīwàng rénrén dōu yǒu.) — Хочется, чтобы у каждого было (без исключения).
```

---

## 4. X 是 + 个 + Y — Basic "X is a Y" Structure

Базовая структура определения через счётное слово 个:
```
脑子是个好东西。 (Nǎozi shì ge hǎo dōngxi.) — Мозг — хорошая штука.
```
