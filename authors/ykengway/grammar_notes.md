## 1. USED TO vs WOULD (past habits/states)

| Form | Use | Example |
|---|---|---|
| **used to** | прошлые состояния/привычки, которых больше нет (включая states: know, like, be) | He used to have long hair. |
| **would** | повторяющиеся действия в прошлом (только actions, НЕ states!) | Every summer, we would go to the seaside. |

**⚠️ Key rule:** *would* нельзя использовать для состояний (know, like, be) — только *used to*.
```
✅ I used to like spinach.
❌ I would like spinach. (неправильно для state)
```

## 2. USED TO / BE USED TO / GET USED TO — Full Comparison

| Form | Structure | Meaning | Example |
|---|---|---|---|
| **used to** | + infinitive | прошлая привычка, которой больше нет | I used to smoke. |
| **be used to** | + noun/-ing | состояние привыкшего человека | I'm used to waking up early. |
| **get used to** | + noun/-ing | процесс привыкания | I'm getting used to the new schedule. |

**Cheat sheet:**
- used to DO = делал раньше, не делаю сейчас
- be used to DOING = привычен к этому сейчас
- get used to DOING = привыкаю к этому (процесс)

## 3. "Myth of Perfect Grammar"

Главная мысль: **ясность и уверенность важнее идеальной грамматики**. Носители языка регулярно "нарушают" правила в casual речи:

```
"Me and Sarah went to the store." (грамматически правильнее: "Sarah and I", но это очень частая разговорная форма)
"I don't know nothing." (double negative — нестандартно, но встречается в некоторых диалектах)
then vs than (постоянно путают даже носители)
```

**Takeaway:** Не нужно бояться говорить из страха сделать ошибку — понятность и уверенность в разговоре важнее безупречной грамматики.
